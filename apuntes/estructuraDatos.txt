R es un lenguaje de programacion orientado a objetos
Este lenguaje esta organizado alrededor de objetos. Todo en R es un objeto
En programacion se trabaja con estructura de datos para almacenar los datos y funciones para almacenar procesos.
Una estructura de datos es una interface para organizar la memoria de la computadora.
R no provee acceso directo a la memoria de la computadora pero provee un gran numero de estructura de datos para 
referencia los objetos.
Cada estructura de datos esta diseñada para optimizar los aspectos de almacenamiento, acceso y procesamiento.

*Atomic Vector
*List

*Array
